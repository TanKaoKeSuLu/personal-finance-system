# 💰 Personal Finance System

一个基于 Spring Boot + Vue3 的个人记账管理系统。

## 项目介绍

本项目用于记录个人日常消费，
支持账单管理、数据统计、用户管理等功能。

## 技术栈

### 后端
- Spring Boot
- MyBatis
- MySQL
- JWT
- BCrypt
- EasyExcel

### 前端
- Vue3
- Vite
- Element Plus
- Axios
- ECharts

## 已实现功能

✅ 用户注册登录

✅ JWT身份认证

✅ BCrypt密码加密

✅ 账单新增/删除/修改/查询

✅ 条件搜索

✅ 分页展示

✅ Excel导出

✅ 消费分类统计

✅ 月度消费统计
